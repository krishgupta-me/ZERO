The goal of this repository is to create, test, and iterate quickly while documenting the process along the way.

## Getting Started

### Prerequisites

```bash
Node.js >= 18
npm or pnpm
```

### Installation

```bash
git clone https://github.com/krishgupta-me/ZERO.git

cd ZERO

npm install
```

### Run Locally

```bash
npm run dev
```
